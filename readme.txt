На главной странице есть список групп, чтобы посмотреть расписание нужно выбрать определенную группу.
Для того чтобы добавлять и редактировать расписание, нужно нажать на кнопку "Я методист" и авторизоваться, после чего у вас появятся кнопки: "Добавить предмет", "Добавить аудиторию", "Добавить преподавателя", "Добавить расписание", "Добавить время пар".
Чтобы добавить расписание нужно сначала добавить все предметы, аудитории и всех преподавателей. 
У каждого преподавателя есть предметы, которые он может преподавать, их надо указать при добавлении преподавателя.
После добавления всех выше перечисленных пунктов можно добавлять расписание. Система не даст заполнить не логичное расписание. При добавлении расписания нужно указать дату, время, предмет, преподавателя, группу и аудиторию.
После добавления расписание появится само, при этом его можно удалить, если вы обладаете правами методиста.
